# apache-mirror
Easily mantain an Apache mirror updated twice a day with this Docker image.

## Usage
Clone this repository
```bash
git clone https://github.com/brunneis/apache-mirror
cd apache-mirror
```

Launch it!
```bash
docker-compose up -d
```

> All the data will be stored in `./data` by default.
